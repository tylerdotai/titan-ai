## About

Titan AI runs large language models on your own hardware for code generation. No API calls, no data leaves your machine.

## Features

- Local inference with Ollama
- FastAPI backend
- Code generation endpoints
- GPU-accelerated (CUDA/ROCm)
- Web interface

## Stack

Python • FastAPI • Ollama • Next.js • TypeScript
